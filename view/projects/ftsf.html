FTSF-Webshop
Fair Trade Software

There was asked to build a webshop for the Fair Trade Software Foundation where multiple kinds of merchandise get offered. For this project, I was responsible for multiple pages to make a desktop version and the mobile version. I created all the pages with the delivered design and build it to a dynamic website.

I was also responsible for building the whole homepage and product page where there had to be a couple of elements implemented like dropdowns and special styling for the design. I used Sass/SCSS and HTML for the styling and used BEM to create a structured project. All the data I processed with PHP and JavaScript.
